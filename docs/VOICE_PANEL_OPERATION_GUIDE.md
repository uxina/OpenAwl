## 🎮 语音面板配置逻辑 (voice-panel-config.js)

### 核心设计理念

`voice-panel-config.js` 是语音面板的核心配置文件，采用**分步引导**的设计理念，将复杂的夜间流程拆分为独立的步骤，主持人可以：
1. **手动控制每一步** - 确保玩家有充足时间执行动作
2. **自动播放辅助步骤** - 如"放下大拇指"等提示可自动播放
3. **灵活应对不同人数** - 5-7人局和8-10人局使用不同的夜间流程

### 配置结构

```javascript
VoicePanelConfig = {
    // 音频基础路径
    audioBasePath: '/audio/commands',
    
    // 数字音频映射（用于拼接播报）
    numberAudioMap: { 0-10: 'numbers/NUM-X.mp3' },
    
    // 语音片段（用于拼接）
    segmentAudioMap: { room-created, person, round, leader-is, ... },
    
    // 完整的语音指令映射表
    voiceCommands: { CMD-001 ~ CMD-156 },
    
    // 夜间步骤配置（核心）
    nightSteps: { '5-7': [...], '8-10': [...] },
    
    // 队伍配置
    missionConfig: { 5: [2,3,2,3,3], 6: [...], ... },
    
    // 阶段到默认语音的映射
    phaseVoiceMap: { waiting, night, day, voting, ... },
    
    // 面板按钮到语音的映射
    panelButtonVoiceMap: { vote-passed, mission-success, ... }
}
```

### 夜间流程设计（分步控制）

### 最有效的游戏推进方式

#### 1. 开场阶段（快速启动）

**推荐流程**:
```
CMD-001 (开场致辞) 
→ CMD-003 (角色配置说明) 
→ CMD-004 (创建房间) 
→ 等待玩家加入
→ CMD-008 (全员准备) 
→ CMD-011 (进入夜间)
```

**优化建议**: 
- 可以跳过 CMD-002 (详细规则)，除非玩家要求
- CMD-004 后播放房间号语音，方便玩家加入

#### 2. 夜间阶段（分步精确控制）

**核心原则**: 每个动作单独一步，主持人观察玩家完成后再下一步

**标准操作**:
1. 播放指令
2. 等待玩家执行（观察玩家动作）
3. 确认完成后点击下一步
4. 重复直到夜间结束

**关键步骤**:
- CMD-N02 (坏人睁眼): 必须确认坏人互认完成
- CMD-N05 (梅林睁眼): 必须给梅林足够观察时间
- CMD-N09 (派西维尔睁眼): 必须给派西维尔足够分辨时间

#### 3. 白天/组队阶段（灵活推进）

**推荐流程**:
```
CMD-041 (白天开始) 
→ CMD-042 (队长任命) 
→ 自由讨论 (CMD-048)
→ CMD-052 (讨论结束)
→ CMD-044D (选择队员)
→ CMD-071 (投票开始)
```

**优化建议**:
- 讨论时间由主持人控制，可长可短
- 使用 CMD-053D (催促选择) 加速慢速玩家

#### 4. 投票阶段（快速确认）

**推荐流程**:
```
CMD-071 (投票开始)
→ 等待所有玩家投票
→ CMD-074 (通过) 或 CMD-075 (否决)
→ 如果否决次数多，播放 CMD-083 (连续否决提示)
```

#### 5. 任务阶段（结果导向）

**推荐流程**:
```
CMD-091 (任务开始)
→ 等待队员选择
→ CMD-094 (成功) 或 CMD-095 (失败)
→ CMD-103 (当前比分)
```

**特殊规则**:
- 第4轮播放 CMD-101 (需要2个失败才失败)
- 第5轮播放 CMD-105 (最终轮提示)

#### 6. 刺杀阶段（紧张氛围）

**推荐流程**:
```
CMD-111 (进入刺杀)
→ CMD-112 (刺杀开始)
→ 等待刺客选择
→ CMD-114 (确认刺杀)
→ CMD-115 (执行刺杀)
→ CMD-116 (成功) 或 CMD-117 (失败)
```

### 最佳实践建议

1. **夜间流程**: 严格按照分步执行，不要跳过步骤
2. **讨论时间**: 根据玩家活跃度灵活控制，活跃玩家多给时间
3. **投票催促**: 超过30秒未投票时播放催促语音
4. **比分播报**: 每轮任务结束后播报比分，让玩家了解局势
5. **身份提示**: 夜间阶段适当播放角色提示（CMD-122, CMD-123）增加氛围
