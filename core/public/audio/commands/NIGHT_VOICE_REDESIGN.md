## 音频后处理

生成后使用FFmpeg标准化：

```bash
for file in CMD-016.mp3 CMD-017.mp3 CMD-017B.mp3 CMD-018A.mp3 CMD-018B.mp3 CMD-018C.mp3 CMD-018D.mp3 CMD-019A.mp3 CMD-019B.mp3 CMD-019C.mp3 CMD-019D.mp3 CMD-019E.mp3 CMD-020.mp3 CMD-020B.mp3 CMD-022.mp3; do
    ffmpeg -i "$file" \
        -acodec libmp3lame \
        -ar 48000 \
        -ac 1 \
        -b:a 65k \
        -af "loudnorm=I=-14:TP=-1.5:LRA=11,afftdn=nf=-25" \
        "${file}.tmp.mp3"
    mv "${file}.tmp.mp3" "$file"
done
```

## 设计原则

1. **每个操作单独一步**: 睁眼、闭眼、竖大拇指、放下大拇指都是独立步骤
2. **留出足够时间**: 每步之间有过渡，让玩家有足够时间执行动作
3. **清晰明确**: 语音指令明确告诉玩家应该做什么
4. **符合标准流程**: 按照阿瓦隆官方规则设计的夜间流程
